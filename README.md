# Niyog & Coir

Eco-friendly coconut coir products — Philippine-made, 100% natural.

## Project Setup

```bash
npm install
```

### Development
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

### Deploy to GitHub Pages
```bash
npx gh-pages -d dist
```
